1 · What Scroll Sense Is

Scroll Sense is a digital wellness app that helps users reduce mindless scrolling and build healthier routines through:

  • Morning planning
  • Evening reflection
  • App blocking and focus modes
  • Mood, sleep, and journal tracking
  • Gamification (XP, gems, streaks, badges, leagues)
  • AI-powered guidance and content analysis

The app is designed to help users understand their social media habits and replace passive screen time with intentional actions.

2 · Core User Experience

🏠 Home Hub

Users land on a central home screen that includes:

  • Personalized greeting
  • Streak and XP progress indicators
  • Quick access cards for Daily Check-In, Goals, Mindful, Sleep, Journal, Report, Rewards, Pomodoro, and App Blocking
  • "Plan Your Day" preview with completion toggles
  • "Quick Reflection" widget (morning intention / evening check-in prompt)
Why this helps: users can see progress and jump straight to the next best habit in seconds.

🧭 Bottom Navigation

Fast tab-level navigation supports Dashboard, Report, Mindful, Journal, and More — plus direct pathways to Goals, Rewards, Sleep, Pomodoro, Plan Your Day, and App Blocking via cards and buttons.

Why this helps: less friction, faster consistency.

🗺️ Guided Flows

The app includes onboarding tours and feature-level guided tours so users understand what to do first.

Why this helps: first-time users are less overwhelmed and activate value faster.

5 · FAQs

Getting Started

What is Scroll Sense?
A digital wellness app that helps users reduce mindless social media use through planning, reflection, app blocking, focus tools, and rewards.
Do I need an account?
You can try guest mode, but account mode gives stronger persistence and sync.
How does the daily flow work?
Morning: Social Media Time Planner. Evening: Daily Check-In.

Goals, Planner, and Tasks

Can I create my own goals?
Yes, including timing-based and social media goals.
Can I edit or delete goals later?
Yes. Goals can be edited and removed.
Can I share my daily plan with parents?
Yes. Saved plan entries can be shared using the share button / share sheet.

App Blocking

Can I block different apps for different contexts?
Yes. Separate selections exist for default, Pomodoro, goals, mindful, and sleep contexts.
Why is an app blocked right now?
Blocking can be triggered by active contexts such as sleep, goals, Pomodoro, mindful sessions, or the default schedule.
What if app blocking seems off?
Screen Time / Family Controls permissions and active schedules / selections must be set.

Mindfulness and Sleep

What is auto mindful nudge?
A reminder that can trigger when usage thresholds are crossed.
Can I block apps after bedtime?
Yes, with configurable sleep boundaries.

Data and Privacy

Do users keep data in guest mode?
Guest mode is intentionally limited compared to signed-in persistence.
Can users delete accounts?
Yes, account deletion is supported.

Rewards

How do users earn XP and gems?
By completing healthy actions across focus, reflection, planning, and consistency features.
Can users compete with others?
Yes, leaderboard and league mechanics are present.

6 · How-To Guide

Everything you can create, edit, delete, view, and share — by action.

  • Create a goal: open Goals → add goal → choose type / options → save.
  • Create a to-do: open To-Do section → add task details → set reminder / priority → save.
  • Create a day-plan entry: open Plan Your Day → add time blocks → save entry.
  • Create a journal entry: open Journal → add title / content / mood / tags → save.
  • Create a sleep log: open Sleep → fill bedtime / wake / quality / notes → save.
  • Create social media planner targets: open Social Media Time Planner → set category targets → save.
  • Edit goals: open goal details → modify time / type / target / blocking options → save.
  • Edit to-dos: open task → change due date / priority / reminder / text → save.
  • Edit day-plan entries: open saved entry → modify blocks / times → save.
  • Edit journal entries: open entry → update content / mood / tags / privacy → save.
  • Edit sleep logs: open entry from history → adjust data → save.
  • Edit profile / avatar / theme: open Settings → update profile / photo / avatar / background / font / theme options.
  • Delete goal: open goal actions / menu → delete.
  • Delete to-do: swipe or open task action menu → delete.
  • Delete day-plan item / entry: open plan item / entry actions → delete.
  • Delete journal entry: open entry actions → delete.
  • Delete sleep entry: open sleep history item actions → delete.
  • Delete account: open Settings → account options → delete account (confirmation required).
  • View check-in history: Daily Check-In history / calendar view.
  • View planner history: Social Media Time Planner history view.
  • View focus trends: Dashboard insight cards + Scroll Score.
  • View report analysis: Report screen result sections after text / screenshot analysis.
  • View progress: Rewards tabs (Progress, Streaks, Badges, Challenges, League, Leaderboard).
  • View Pomodoro history: Pomodoro history section.
  • View sleep trends: Sleep history / stats.
  • Share Plan Your Day entry: open saved plan entry → tap share button.
  • Share friend / competition identity: use leaderboard friend code flow.
  • Share support feedback: via support / email pathways in Settings.
